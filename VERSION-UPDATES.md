# 版本更新记录

## 2026-06-09

### 记账
- 记账表单调整为第一行放“收入/支出 + 分类 + 金额”，备注单独一行。
- 分类下拉框缩短，适合 2-4 个字的分类名。
- 新增“不分类”，默认记账不再自动猜分类；选择过分类后，下次记账继续默认该分类。
- 默认分类也可以从分类列表删除；删除分类不会删除已经记过的历史账。
- 分类管理改到分类下拉里：底部 `＋` 新增分类，左右滑动分类项可露出红色删除按钮。
- 记账统计改为“日历 + 日/周/月/年”，可按选中日期所在的日、周、月、年统计。
- 记账按钮和日历/统计范围合并到一行。
- 记账记录支持修改日期、修改分类、双击备注修改内容，并支持左滑删除。

### 界面
- 移除顶部“全屏”按钮，手机和平板优先使用自适应布局。
- 应用名称改为 `memo`。
- 优化日语界面文案，减少机翻感。
- 调整颜文字：按已确认的 8 个颜文字统一替换成功、完成、删除、错误、空状态和输入占位。
- 升级 PWA 缓存版本，减少桌面 Web App 继续显示旧文件的情况。
- 提醒页底部输入框改成更柔和的圆角输入样式。
- 无过期待办时隐藏过期待办区块。
- 已完成待办行的背景色调浅，减少整行色块感。
- 手机和平板禁用页面缩放，Web App 默认铺满屏幕，内部内容区域保持滚动。
- 设置页“功能显示”改成响应式左右列，宽屏更紧凑，窄屏自动变一列。
- 左滑删除增加右滑收回，误划后可直接滑回，不需要点别处。
- 记账分类下拉框改为固定区域内上下滚动，分类项加高，文字更容易看清。
- 待办完成按钮改成更轻的小圆形按钮。
- 电脑网页铺满浏览器；安装成 Web App 后会尝试记住上次窗口尺寸。
- 设置页语言选择移动到底部；系统通知设置默认开启，仍需浏览器授权才能发系统通知。
- 手机底部待办输入栏改为独立提交表单，键盘回车优先显示发送并直接记录，不再跳到旁边的中途提醒下拉框。
- 待办页底部“中途提醒”下拉缩小，把主要宽度留给输入栏。
- 已完成待办支持左滑删除当前日期的完成记录。
- 输入栏新增自然时间识别：支持 `3小时后提醒我xx`、`3h后设定xx`、`半小时后xx`、`今晚8点xx`、`明晚8点xx`、`十天后xx` 等常见写法。
- 记账页收入/支出改为自定义小菜单，样式和分类选择保持一致；分类菜单保持竖向列表，内部可上下滑动，并加宽以完整显示分类文字。
- 新增“菜谱”功能：可记录菜名、食材和做法，支持搜索关键词筛选所有菜谱；设置页可显示或隐藏菜谱入口。
- 新增“经期”功能：可记录开始日期、结束日期和备注；根据最近周期做加权平均预测下次经期、经期长度、排卵日、易孕期、未来几次和可信度；设置页可显示或隐藏经期入口。

### 便签与日记
- 便签支持多个便签、标题编辑和右侧便签列表。
- 便签列表和编辑区左右调换，便签列表放左侧。
- 便签增加删除按钮；只剩一个便签时，删除会清空当前便签。
- 便签列表支持收起/展开。
- 日记页改为单栏布局，日期选择固定在顶部。
- 移除便签和日记页面上的自动保存提示，说明改放到设置里的使用说明。

## 2026-06-08

### 待办与提醒
- 今日待办行按钮由对号改为“完”，避免被误认为普通勾选框。
- 待办输入栏的完成时限文案改为“中途提醒”，减少误解。
- 新增设置页“使用说明”入口，说明如何添加待办、完成、删除、标记 `ing`、添加未来待办和定时提醒。
- 今日待办和进行中的普通事项支持双击编辑文字，回车或失焦保存，Esc 取消。
- 提醒页底部增加输入框，可直接录入每日、每周、每月、明天/后天/大后天等提醒。
- 每天打开后的自定义欢迎弹窗仍保持每日只提示一次。
- 待办输入栏新增可选“中途提醒”，默认不需要；选择 1/2/4/8 小时时，会从添加时间开始计算，并在中间时段弹一次提醒。

### 设置
- 日记密码输入仅在开启四位数字密码后显示，输入框不再显示提示文字。
- 新增“功能显示”设置，可分别隐藏不常用的便签、日记和记账入口。
- 新增独立日记入口，日记和便签分开保存。
- 日记支持开启四位数字密码；已有密码时，修改或关闭密码必须先验证原密码。
- 顶部日期保留星期，并使用日语星期字符：日、月、火、水、木、金、土。

### 记账
- 新增默认分类：吃饭、购物、交通、办公、工资、报销、其他。
- 支持自定义记账分类；无记录的自定义分类可删除。
- 记账记录显示分类，旧记录未设置分类时按“其他”展示。

### 界面
- 增加更柔和的按钮、分类标签、行内编辑和说明弹窗样式。
- 底部输入框继续保持手机端更明显。
- 电脑端默认改成窄版小看板；手机和平板优先保持铺满和自适应。
