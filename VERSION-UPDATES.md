# 版本更新记录

## 2026-06-09

### 记账
- 记账表单调整为第一行放“收入/支出 + 分类 + 金额”，备注单独一行。
- 分类下拉框缩短，适合 2-4 个字的分类名。
- 默认分类也可以从分类列表删除；删除分类不会删除已经记过的历史账。
- 今日、昨日、本月、上月筛选默认回到“今日”，不再记住上一次筛选。

### 界面
- 提醒页底部输入框改成更柔和的圆角输入样式。
- 无过期待办时隐藏过期待办区块。
- 待办完成按钮改成更轻的小圆形按钮。
- 电脑网页铺满浏览器；安装成 Web App 后会尝试记住上次窗口尺寸。

### 便签与日记
- 便签支持多个便签、标题编辑和右侧便签列表。
- 便签列表支持收起/展开。
- 移除便签和日记页面上的自动保存提示，说明改放到设置里的使用说明。

## 2026-06-08

### 待办与提醒
- 新增设置页“使用说明”入口，说明如何添加待办、完成、删除、标记 `ing`、添加未来待办和定时提醒。
- 今日待办和进行中的普通事项支持双击编辑文字，回车或失焦保存，Esc 取消。
- 提醒页底部增加输入框，可直接录入每日、每周、每月、明天/后天/大后天等提醒。
- 每天打开后的自定义欢迎弹窗仍保持每日只提示一次。
- 待办输入栏新增可选“完成提醒”，默认不需要；选择 1/2/4/8 小时完成时，会从添加时间开始计算，并在中间时段弹一次提醒。

### 设置
- 新增“功能显示”设置，可分别隐藏不常用的便签、日记和记账入口。
- 新增独立日记入口，日记和便签分开保存。
- 日记支持开启四位数字密码；已有密码时，修改或关闭密码必须先验证原密码。
- 顶部日期保留星期，并使用日语星期字符：日、月、火、水、木、金、土。

### 记账
- 新增默认分类：吃饭、购物、交通、办公、工资、报销、其他。
- 支持自定义记账分类；无记录的自定义分类可删除。
- 记账记录显示分类，旧记录未设置分类时按“其他”展示。

### 界面
- 增加更柔和的按钮、分类标签、行内编辑和说明弹窗样式。
- 底部输入框继续保持手机端更明显。
- 电脑端默认改成窄版小看板，顶部新增全屏按钮；手机和平板仍保持铺满屏幕。
